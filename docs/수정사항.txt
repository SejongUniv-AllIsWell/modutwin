ocs수정사항

##
basemap 업로드 후  업로드 내역에서 상태가 문 꼭짓점 지정으로 되어있고, 정합하기버튼이 살아있는것 수정. 
-> basemap 문지정 완료 후 업로드까지 완료 되면 상태는 업로드 완료. 관리자가 basemap등록하면 상태는 "Basemap"으로 변경. 정합하기 버튼은 없어야 함.

##
현재는 관리자가  basemap관리에서 승인을 누르면 "활성화" 버튼까지 눌러야 하지만 승인을 누르면 등록이 되도록 수정.
또한, 관리자가 승인을 완료하면 건물 / 층/ 모듈 표시에서 따로 표시 버튼을 누르지않아도 되도록 상태 자동으로변경하여 표시되도록 수정

##
문 열었을 때 두 문 사이 공간이 비어있음 

##
모듈 정합단계에서  업로드는 완료후 이탈 시, 랜딩페이지에서 건물/모듈/기여자에서 모듈의 개수가 1증가(정합완료로인식)하는 현상이 발생. 
원하는 동작은 모듈 정합단계에서 이탈 시 

## 천장 바닥 설정 및 벽면 설정 시
우클릭&드래그로 pointcloud를 이동시켜서 잘 볼 수 있도록 수정. pointcloud이동이 아니라 뷰 이동이어도 괜찮음.

##
buildings/{building_id}페이지의 좌측 패널에서 건물이름아래의 "FLOORS(*)"를 삭제.
buildings/{building_id}/fllors/{floor_id} 페이지의 좌측 패널에서 "ROOMS(*)"를 삭제.

## 
랜딩페이지 시작하기 -> explore페이지 건물목록에서 건물 선택 시 buildings/{building_id}페이지로 이동함. 이때, basemap등록 가능하고, buildings/{building_id}/fllors/{floor_id} 페이지에선 module등록 가능함. 하지만 이는 ply업로드에 해당하는것이고 현재, 영상업로드와 images.zip+colmap 업로드를 실제로 수행할 수 있는 루트가 없음. 따라서 기존의 basemap등록과 module등록 울 선택할 시 선택 페이지로 리다이렉트. 이 선택 페이지는 register페이지로 새로운 페이지야. 영상 등록, 3DGS 에셋 등록, 이미지+SfM 등록  버튼이 있고, 선택할 수 있으며, 영상 등록 버튼을 누르면 register/video 페이지로 이동하여 드래그앤드롭할 수 있는 섹션이 생기고, 이 섹션은 버튼이야. 버튼을 클릭하여 사용자 파일 시스템에서 선택 할 수 있게 하거나, 드래그앤드롭으로 비디오 파일을 업로드할수 있어. 3DGS에셋 등록은 register/assets페이지에서 현재 basemap, module등록 각 페이지를 그대로 사용하도록하고, 이미지+SfM등록 페이지는 register/images 페이지에 비디오 업로드처럼 zip파일을 받을 수 있게 해. 각 페이지에서 업로드한 파일 검증 수행과정도 없으면 만들어.

##
랜딩페이지의 "기록을 추가하는 방법" 섹션에서 "영상 업로드"를 "영상 등록"으로 수정 하고 세부 내용 중 "학습이 끝나는 즉시 아틀라스의 일부가 됩니다." 를 제거 ,"포인트 클라우드 제출"을 "3ㅇDGS 에셋 등록"으로 수정, "이미지+SfM 기여"를 "이미지+SfM 등록"으로 수정 하고 세부 내용 중 "OpenSfM 결과물을 가져오세요." 를 "SfM 결과물을 가져오세요"로 수정
